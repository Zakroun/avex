# AVEX – Modern Streetwear E-Commerce

AVEX is a modern fashion e-commerce web application focused on streetwear culture. The platform allows users to explore and shop premium products such as hoodies, jeans, and sneakers through a clean and responsive interface. The project emphasizes performance, modern UI design, and smooth user experience.

## Features

* Modern and responsive homepage
* Product categories (Hoodies, Jeans, Sneakers)
* Featured products section
* Clean and minimal UI design
* Smooth CSS animations and interactions
* Scalable component-based architecture

## Tech Stack

* React
* Vite
* CSS
* JavaScript

## Getting Started

### 1. Clone the repository

git clone https://github.com/yourusername/avex.git

### 2. Navigate to the project

cd avex

### 3. Install dependencies

npm install

### 4. Run the development server

npm run dev

The application will start on a local development server.
